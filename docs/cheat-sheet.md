## Soundbites to Have Ready

**Why custom?**
> "Could you ask regular Copilot to summarise this grant? Sure. But would it check each criterion against your org profile one by one? Would it know that the right answer is 'check with Finance' instead of guessing? That's what makes this a custom agent."

**Five key messages:**
1. You don't need to be technical — you need to understand the problem
2. Start vague, end sharp — the agent helps you get there
3. Human-centered — the agent centers the person, not the tech
4. Copilot, not replacement — human judgment stays in the loop
5. This output is your foundation — Stage 2 and 3 build on it
